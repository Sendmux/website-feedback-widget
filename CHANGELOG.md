# Changelog

All notable changes to this project will be documented here.

## 0.1.2 - 2026-05-26

- Added a centred modal position for in-app feedback flows.
- Added configurable minimum message length validation with accessible inline errors.
- Redesigned the widget flow with category, compose, success, and error screens.
- Replaced tight step screenshots with a complete all-screens README preview.

## 0.1.1 - 2026-05-26

- Fixed hidden status messages so success and error rows do not appear before submission.
- Refreshed README previews with widget-only category and message-step screenshots.

## 0.1.0 - 2026-05-25

- Initial Sendmux Feedback Widget package.
- Added custom element embed, global helper API, webhook delivery, and secure relay documentation.
- Added accessible modal form, responsive fixed positioning, dark mode, and powered-by branding.
